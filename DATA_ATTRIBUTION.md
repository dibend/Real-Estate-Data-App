# Data Attribution

This project is an unofficial dashboard built on top of Zillow Research housing data.

Primary source:
- Zillow Research Housing Data: https://www.zillow.com/research/data/

Metric guidance:
- Zillow Home Value Index (ZHVI) User Guide: https://www.zillow.com/research/zhvi-user-guide/

Suggested attribution for this project:

`Source: Zillow Research Housing Data, Zillow Home Value Index (ZHVI), accessed via https://www.zillow.com/research/data/.`

Important usage notes:
- ZHVI should be described as the typical home value for a region, not the median home value.
- Zillow notes that data is updated monthly and that CSV download paths may occasionally change.
- This repository does not grant any rights to Zillow's data beyond the rights provided by Zillow.
- The software in this repository is licensed separately from the data files used with it.

This project is not affiliated with, endorsed by, or sponsored by Zillow.
