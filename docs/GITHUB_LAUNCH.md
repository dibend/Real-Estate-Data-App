# GitHub Launch Notes

## Suggested repository description

`ZIP-level U.S. housing market dashboard built on Zillow Research ZHVI data, Quart, Pandas, and Plotly.`

## Suggested repository topics

`real-estate`, `housing-market`, `zillow`, `zhvi`, `usa`, `plotly`, `quart`, `pandas`, `data-visualization`

## Suggested pinned README message

Use the dashboard to compare ZIP codes, scan statewide movers, and browse county-level price trends without digging through raw CSV files.

## Suggested sponsor call-to-action

If this project saves you research time, helps with local market analysis, or gives you a solid open-source housing dashboard base, consider sponsoring ongoing maintenance through GitHub Sponsors.

## GitHub repo checklist

1. Add the social preview image in repository settings.
2. Paste the description and topics above into the repo metadata.
3. Enable GitHub Sponsors for the owner account if it is not already active.
4. Keep `.github/FUNDING.yml` aligned with the sponsoring account.
5. Create the first release after the README, screenshots, and setup steps are final.
